# ParkWiz — ערכת פיילוט עירונית (Pilot Kit)

**מה זה:** שכבת אנליטיקה שהופכת מצלמות CCTV עירוניות קיימות לחיישני תפוסת-חניה ואכיפה — בלי חומרה חדשה, בלי החלפת Pango/Cellopark.

## הצינור (הכל בנוי ועובד)
```
מצלמת IP קיימת → frame-grabber.js (snapshot כל 10שנ')
                → vision-bridge.js (Gemini Flash Vision → JSON תפוסה,
                   עם חסימת-הזיות + debounce ריצודים)
                → occupancy.jsonl
                → pilot-dashboard.html (עברית RTL: מפת תפוסה חיה,
                   גרף לאורך זמן, KPI, כפתור "הפק דוח עירוני")
```

## הפעלה
- **דמו (בלי מצלמה):** שים תמונה ב-`vision-in\frames\latest.jpg` והרץ `bin\run-pilot.cmd`
- **חי (עם מצלמה):** `bin\run-pilot.cmd http://user:pass@camera-host/snapshot.jpg`
- פתח `pilot-dashboard.html` בדפדפן — נטען עם הדאטה האמיתית (occupancy.json), ואם אין — דמו מובנה.

## ה-Pitch לעירייה (מאומת מול מחקר Perplexity, 45 מקורות)
- **למי:** מנהל אגף אכיפה+חניה (בעל-עניין) + מוביל עיר-חכמה (ספונסר, דרך מוקד 106)
- **מה:** פיילוט 3-6 חודשים על רחוב אחד, 30-50 אלף ₪ — מתחת לרף פטור-ממכרז (לאמת מול גזבר), או חינם עם MOU
- **ישות:** עוסק מורשה + ביטוח אחריות מקצועית מספיקים לפיילוט (בע"מ רק לפרודקשן)
- **בידול:** Wisesight/Pumba מתקינים חיישנים ($$$); אנחנו אפס-חומרה על תשתית קיימת. Pango = תשלום, לא תפיסה — אנחנו משלימים אותם
- **KPIs לפיילוט:** דיוק תפוסה מול פקח, הפרות שזוהו לשעה, הכנסת אכיפה תוספתית

## פרטיות (מובנה)
לא שומרים וידאו; מחלצים מטא-דאטה בלבד (מספרים: תפוס/פנוי). בלי זיהוי פנים/לוחיות בפיילוט. עומד ברוח תיקון 13 לחוק הגנת הפרטיות.

## מגבלות ידועות (ביושר)
- דיוק ה-vision תלוי זווית/תאורה — לכן debounce + חסימת-הזיות + כיול פר-מצלמה (MAX_SPOTS)
- מפתח Gemini חינמי = מגבלת קצב; לפיילוט חי נדרש מפתח ייעודי (עלות ~אפסית)
- דורש הרשאת גישה טכנית ל-snapshot של מצלמה עירונית אחת (או RTSP→snapshot דרך ה-VMS)
